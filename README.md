# 🎮 Rock, Paper, Scissors (Python)

## Description

A simple Rock, Paper, Scissors game I built using Python. The player goes against a computer that randomly picks its move each round. This project helped me practice functions, loops, conditionals, user input, and using Python's `random` module.

---

## Features

* Play against an AI opponent
* Random computer choices
* Best-of-7 gameplay (first player to 4 wins)
* Score tracking after each round
* Input validation for invalid choices
* Main menu to play again or quit

---

## Technologies Used

* Python 3
* Python `random` module

## How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python rock_paper_scissors.py
```
